# テトリス＆カードゲーム

テトリスをプレイし、ゲームオーバー後にカード引きフェーズで底辺を消すことを目指す2フェーズのWebゲームです。

## 遊び方

### フェーズ1: テトリス

通常のテトリスをプレイします。ブロックが天井に到達するとゲームオーバーになり、そのボード状態のままカードフェーズへ移行します。

### フェーズ2: カードフェーズ（本番）

テトリス終了後のボードを使って、**底辺の列を消す**ことを目指します。

1. 必要に応じてプールから除外したいアイテムをトグルで外す
2. **START** ボタンを押してカードを引く
3. 引いた結果に応じてアクション：

| 引いた結果 | アクション |
|---|---|
| I / O / T / S / Z / J / L | そのピースが出現、**3秒以内**に配置する |
| **ACE** | 好きなピースを選んで配置できる |
| **JOKER** | そのターンをスキップ |

4. 横一列が埋まるとライン消去（テトリスと同様）
5. **最下段（底辺）の列が消えたらクリア！**

## 操作方法

| キー | 動作 |
|---|---|
| `←` `→` | ピースを左右に移動 |
| `↑` / `Z` | ピースを右回転 |
| `X` | ピースを左回転 |
| `↓` | ソフトドロップ（ゆっくり落とす） |
| `Space` | ハードドロップ（即座に落とす） |

### スマートフォン

| タッチ操作 | 動作 |
|---|---|
| タップ | 回転 |
| 左スワイプ | 左移動 |
| 右スワイプ | 右移動 |
| 下スワイプ | ハードドロップ |

## 技術仕様

- **言語**: Vanilla JavaScript / HTML5 / CSS3
- **描画**: HTML5 Canvas API
- **依存ライブラリ**: なし（外部リソース不要）
- **動作環境**: 最新のChrome / Firefox / Safari / Edge

## ゲームロジック詳細

- **ランダマイザー**: 7-bag方式（テトリスガイドライン準拠）
- **回転システム**: SRS（Super Rotation System）壁蹴り対応
- **タイマー**: Phase2の配置制限時間は3秒（SVGリングで視覚的にカウントダウン）
- **除外機能**: STARTを押す前にプールから任意のアイテムを除外可能（最低1つは残る）
